Theme Features

  • 22 color options
  • customizable post sizes
  • customizable post padding
  • customizable font size
  • customizable photoset gutter width
  • customizable post margins
  • “Like” and “reblog” buttons
  • optional header image
  • optional header welcome text
  • optional header section
  • up to 10 custom links
  • optional photo fade
  • optional monochrome posts hover
  • optional smooth scrolling
  • search bar & manually inserted search suggestions
  • sticky sidebar

Header image:

  • the height is manually inserted in the “header image height” text field the rest of the header dimensions will adjust based on this height
  • a transparent image / render is recommended

Header “welcome” text:

  • the text section is vertically aligned based on your header image height aka it will always be in the center, so don’t worry about the text amount
  • this is different from your own blog’s description

Search bar:

  • search suggestions are manually inserted in the text fields
  • if you don’t want to enable suggestions, clear the text the in the text field
  • you can have up to 5 suggestions
    if you would like to add more, please contact me
  • check out this post for more information

Pagination:

  • “pagination in the sidebar” option ON: self explanatory
    “pagination in the sidebar” option OFF: pagination will appear at the END/bottom of the posts on the current page
